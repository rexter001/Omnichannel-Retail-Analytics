# Member 3 – Customer Segmentation

## Overview
This module performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis and K-Means clustering.

## Dataset
Location:

data/customer_rfm.csv

Features:
- Recency
- Frequency
- Monetary

## Techniques Used
- StandardScaler
- PCA (Principal Component Analysis)
- K-Means Clustering
- Elbow Curve
- Silhouette Score

## Outputs
- Customer Segments
- Cluster Scatter Plot
- Cluster Analysis
- Business Recommendations
